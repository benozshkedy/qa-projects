# PlayStation Web Project – QA Portfolio

This project contains manual test design and execution
performed on the official PlayStation Israel website (he-il).

The focus of this testing effort was:
- Functional validation
- UI behavior
- Localization (RTL Hebrew)
- Navigation and link integrity
- Business logic and region validation

---

## System Under Test

Website: https://www.playstation.com/he-il/  
Locale: Hebrew (RTL)  
Testing Type: System Testing / UI Testing  

---

## Test Scope

### 1. Homepage
- Page load validation
- Navigation menu verification
- Banner redirection
- RTL layout validation
- Mobile responsiveness

### 2. Products – PS5
- Console information accuracy
- Subpage navigation
- Purchase/CTA validation
- Israeli region consistency

### 3. Localization
- Hebrew language validation
- Language switching
- Region persistence across navigation

### 4. Links Validation
- Internal links integrity
- External links behavior
- Broken link detection (404/5xx)
- New tab behavior validation

---

## Test Design Approach

- Positive scenarios
- Negative scenarios
